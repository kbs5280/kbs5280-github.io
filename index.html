Projects
My flagship project for this year is PreQ™. PreQ is a Lending-Tree style web application. It allows our clients to enter a single set of personal information and instantly receive loan pre-qualification offers from as many as 4 lenders.
Under-the-hood, PreQ™ validates, sanitizes and formats data for four unique APIs, including three REST APIs and a SOAP API. Data is formatted as JSON, XML and URLEncoded objects customized to each API. Data is sent via web-standard HTTPS and utilizes OAUTH, MTLS, and AES256 bit encryption to securely transmit data.
The app is built in Salesforce and has a pubically accessible site built on Salesforce Sites. The controllers and API services are built in Apex and utilize SOQL to retrieve data from and save data to an extensive relational database.
PreQ™ is a greenfield project that I was solely responsibly for developing.
