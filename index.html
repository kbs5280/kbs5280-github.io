Projects
My flagship project for this year is PreQ™. PreQ™ is a Lending-Tree style web application. It allows a user to enter a single set of personal information and receive loan pre-qualification offers from four industry-specific lenders.
Under-the-hood PreQ™ validates, sanitizes and formats data for four unique APIs, including three REST APIs and a SOAP API. Data is formatted as JSON, XML and URLEncoded objects customized for each API. Requests are posted via web-standard HTTPS and utilizes OAUTH, MTLS, and AES256 bit encryption to securely transmit data.
The app is built in Salesforce and has a pubicly accessible interface built on Salesforce Sites. The controllers, models and API services are built in Apex and utilize SOQL to retrieve data from and save data to an extensive relational database. The view is built with Visualforce, HTML, CSS and Javascript.
